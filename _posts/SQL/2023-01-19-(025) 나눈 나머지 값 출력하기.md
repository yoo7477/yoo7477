### [SQL 문법]
숫자 값을 나눈 나머지를 출력할 수 있다.

```sql
SELECT MOD(10, 3)
  FROM dual
```
### [예시]
```python
# 오라클 연동 및 접속
import pandas as pd
import cx_Oracle
dsn=cx_Oracle.makedsn('localhost',1521,'orcl')
db=cx_Oracle.connect('scott','tiger')
cursor=db.cursor()

# SQL 문법
cursor.execute("""
SELECT MOD(10, 3)
  FROM dual
""")

row=cursor.fetchall()
colname=cursor.description
col=[]

for i in colname:
    col.append(i[0])

# pandas를 사용한 데이터 프레임 형식으로 변환
emp=pd.DataFrame(row,columns=col)
print(emp)
```
### [결과]
       MOD(10,3)
    0          1
